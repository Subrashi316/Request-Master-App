Help Section

Request Box

Add url to input area, add the full url in the input section using http:// and use the send button to send the request

Request method or type, Add headers and Body type

In Request type you could send different types of request like GET, POST, PUT, PATCH, UPDATE. In add headers section you could add different header options with the request, Below that is the body section where you could send data with body. Only add JSON data to the textarea section to send with the data in case you want to send json or form data

Response Object and Response Data

In Response section there are two section Response Object and the Response Data, The response object or response simply contains the meta data about the current request like the headers, the request type, request method, the request status. Below that is the response which is sent by the server.
